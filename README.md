# Alert Rules

A collection of Prometheus alert rules for monitoring infrastructure and application health.

## Overview

This repository contains pre-configured Prometheus alert rules organized by monitoring target. These rules are designed to detect infrastructure issues, resource constraints, and service failures before they impact users.
